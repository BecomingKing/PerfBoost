# PerfBoost UI 全面改版设计

> 日期: 2026-06-07 | 状态: 已确认

## 概述

重构 PerfBoost 全部前端页面，采用 Windows 11 现代仪表盘风格（深色 + 彩色卡片 + 大数字 + 圆角）。

**改版范围：** 整体布局 + 5 个页面（仪表盘/清理/启动项/进程/设置）

---

## 1. 整体布局

**现状：** TabView 顶部切换 + 顶部大标题，垂直空间浪费  
**改为：** 左侧导航栏 + 右侧内容区 + 底部状态栏

```
┌──────────┬──────────────────────────────────────┐
│  ⚡       │  当前页面标题                          │
│  PerfBoost │                                      │
│           │  内容区（随导航切换）                    │
│ 📊 仪表盘  │                                      │
│ 🧹 清理    │                                      │
│ 🚀 启动项  │                                      │
│ 📋 进程    │                                      │
│ ⚙️ 设置    │                                      │
│           │                                      │
├───────────┴──────────────────────────────────────┤
│ 💻 开机 2h 34m                         v1.0      │
└──────────────────────────────────────────────────┘
```

**技术方案：** 用 `CTkFrame` 替代 `CTkTabview`，左侧放导航按钮，点击切换右侧 Frame 的显示/隐藏

---

## 2. 仪表盘

4 张数据卡片 2x2 网格布局：

| 卡片 | 内容 |
|------|------|
| 🖥 CPU | 使用率百分比 + 进度条 + 温度 |
| 🧠 内存 | 已用/总量 + 百分比进度条 |
| 💾 磁盘 | 所有分区列表，每个有进度条 |
| 🌐 网络 | 上行/下行速率 |

---

## 3. 清理

- 每个垃圾类别用卡片展示：名称 + 大小 + 横条图 + 复选框 + 预览按钮
- 🔒 标记需要管理员权限的类别
- 顶部：扫描按钮 + 一键清理按钮
- 底部状态文字

---

## 4. 启动项

- 每行一张卡片：程序名 + 来源(HKCU/HKLM) + 文件路径 + 开关 + 状态标签
- 影响评估图标：⚠ 文件存在（正常）/ ❌ 文件不存在（可能已卸载）
- 顶部：刷新按钮 + 已禁用列表按钮

---

## 5. 进程

- 表格布局：名称 | PID | CPU% | 内存（带可视化条）| 操作
- 点击选中行高亮，结束进程按钮启用
- 🔒 标记受保护系统进程
- 右键菜单：结束进程 / 打开文件位置
- 底部显示进程总数 + 当前选中项

---

## 6. 设置

两个卡片分组：

- **外观** — 主题模式：跟随系统 / 暗色 / 亮色
- **关于** — PerfBoost v1.0 + Windows 系统性能优化工具

---

## 技术约束

- 框架：CustomTkinter（不换技术栈）
- 主题：默认暗色，支持亮色切换
- 兼容：Python 3.9+
- 不改动 core/ 层，只改 ui/ + app.py

## 文件变更

| 文件 | 操作 | 说明 |
|------|------|------|
| `app.py` | MOD | 左导航 + 右内容布局，底部状态栏 |
| `ui/dashboard.py` | MOD | 4 卡片网格布局 |
| `ui/cleaner.py` | MOD | 卡片式分类列表 |
| `ui/startup.py` | MOD | 卡片式行 + 影响评估图标 |
| `ui/process.py` | MOD | 内存可视化条 + 受保护标记 + 选中统计 |
| `ui/settings.py` | MOD | 卡片分组 + 关于区块 |
