# PerfBoost 品牌标识优化设计

> 日期: 2026-06-08 | 状态: 已确认

## 概述

优化两个品牌/标识体验点：侧边栏底部品牌标识 + 系统托盘图标，提升视觉精致感和信息密度。

---

## 1. 侧边栏底部品牌标识

### 现状

```
───────────
v1.0          ← 灰色小字（字号10，#999/#444），无交互
```

### 改为

```
───────────
┌──────────────────┐
│ ⚡ PerfBoost     │  ← 圆角色块徽章
└──────────────────┘
```

### 规格

- **样式：** 徽章（corner_radius=6），半透明背景（亮色 `#E0E0E0`，暗色 `#1A1A1A`）
- **内容：** "⚡ PerfBoost"（不显示版本号）
- **字号：** 12，颜色比普通文字稍亮
- **交互：** 悬停背景加深 + 光标手型；点击跳转到设置页
- **文件：** `app.py` `_build_sidebar()` 底部的 bottom frame

---

## 2. 系统托盘图标

### 现状

- PIL ImageDraw 画蓝底圆形 + ⚡ emoji 粘文字，64x64
- 无动态状态

### 改为

### 图标绘制

- 尺寸：64x64
- 底色：深蓝 `#1a5276`
- 图形：白色闪电（`ImageDraw.polygon` 手绘多边形）
- 不再使用 emoji 粘字

### 动态状态

| 条件 | 图标颜色 |
|------|---------|
| 正常（CPU ≤ 80% 且 内存 ≤ 85%） | 深蓝 `#1a5276` |
| 告警（CPU > 80% 或 内存 > 85%） | 橙色 `#e67e22` |
| 严重（CPU > 90% 或 内存 > 90%） | 红色 `#e74c3c` |

### 信息展示

- **悬停 tooltip：** "PerfBoost — CPU: 45% | 内存: 62%"（实时更新）
- **右键菜单：** 顶部增加只读状态行展示 CPU/内存

### 文件

- `core/tray.py` — `_create_icon()` 改绘图标 + 增加 `_update_icon()` 动态切换

---

## 技术约束

- 框架：CustomTkinter + pystray + PIL
- 兼容：Python 3.9+
- 不动 core/ 以外的模块（tray 本身就在 core/ 内）
- 图标状态切换通过 `SystemTray._update_icon()` 方法，由 `app.py` 的 `_poll_monitor()` 每轮调用

## 文件变更

| 文件 | 操作 | 说明 |
|------|------|------|
| `app.py` | MOD | 底部 v1.0 → 品牌徽章 + 点击跳转设置 |
| `core/tray.py` | MOD | 重绘图标 + 动态变色 + tooltip + 右键状态行 |
